# 数据库设计

ontology_entity

```sql
CREATE TABLE ontology_entity (

 id UUID PRIMARY KEY,

 code VARCHAR(100),

 name VARCHAR(200),

 version VARCHAR(20),

 status VARCHAR(20),

 created_at TIMESTAMP

);
```

---

ontology_property

```sql
CREATE TABLE ontology_property (

 id UUID PRIMARY KEY,

 entity_id UUID,

 code VARCHAR(100),

 type VARCHAR(50),

 required BOOLEAN

);
```

---

ontology_relationship

```sql
CREATE TABLE ontology_relationship (

 id UUID PRIMARY KEY,

 source_entity UUID,

 target_entity UUID,

 relation_type VARCHAR(50)

);
```
