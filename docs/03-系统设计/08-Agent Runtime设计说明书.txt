# 5. Planner设计

Planner负责：

```text
目标理解

任务拆解

执行规划
```

---

输入：

```json
{
  "goal":"分析最近90天销售下降原因"
}
```

---

输出：

```json
{
  "plan":[
    {
      "step":1,
      "action":"QuerySalesData"
    },
    {
      "step":2,
      "action":"AnalyzeTrend"
    },
    {
      "step":3,
      "action":"GenerateReport"
    }
  ]
}
```

---

接口：

```java
public interface Planner {

   ExecutionPlan createPlan(
      Goal goal
   );

}
```
