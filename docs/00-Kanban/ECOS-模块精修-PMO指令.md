## 八项检查清单

### A. 页面加载 & 控制台
- [ ] browser_navigate 打开页面 → 正常渲染
- [ ] browser_console → 无 error/uncaught
- [ ] 数据加载完成（无永久loading）

### B. CRUD全操作
- [ ] 列表加载 → 200
- [ ] 搜索/筛选 → 过滤生效
- [ ] 新建 → 201 → 列表刷新
- [ ] 查看详情 → 200
- [ ] 编辑 → 200 → 列表刷新
- [ ] 删除 → 200 → 列表刷新
- [ ] 边界测试（空值/超长/SQL注入）

### C. 样式 & i18n
- [ ] 无自定义CSS
- [ ] 复用公共组件
- [ ] 所有中文用 `g()` 包裹，英文翻译完整

### D. 配置项
- [ ] 无硬编码配置值
- [ ] `sys_config` 表有对应配置项
- [ ] 修改配置 → 页面行为变化

### E. 字典
- [ ] 无硬编码选项列表
- [ ] `sys_dict` 有字典 → 前端使用字典API
- [ ] 缺字典条目 → 补全

### F. 依赖
- [ ] 前端跨模块import → tsc通过
- [ ] 后端依赖API → curl可达
- [ ] 端到端联调通过

### G. 交互
- [ ] 按钮有loading/disabling
- [ ] 操作有toast反馈
- [ ] 表单实时校验
- [ ] 空状态有提示
- [ ] 防重复提交

### H. 种子数据
- [ ] 信科场景数据3-5条
- [ ] Flyway迁移文件
- [ ] 重启后可展示
