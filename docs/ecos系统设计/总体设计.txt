# 核心服务设计

---

## ontology-service

职责：

```text
本体管理

关系管理

版本管理
```

接口：

```text
GET /entities

POST /entities

PUT /entities/{id}

DELETE /entities/{id}
```

---

数据模型：

```sql
ontology_entity

ontology_property

ontology_relationship

ontology_version
```

---

## object-service

职责：

```text
对象运行时

对象查询

对象关系
```

---

核心接口：

```text
POST /objects/query

GET /objects/{id}

POST /objects
```

---

对象结构：

```json
{
  "id":"obj-001",

  "type":"Customer",

  "properties": {},

  "relationships": []
}
```

---

