# 1. World Model定位

World Model 是 ECOS 的最高认知层。

如果说：

```text
Ontology
=
企业事实层

Knowledge Graph
=
企业知识层

Agent Mesh
=
企业执行层
```

那么：

```text
World Model
=
企业认知层
```

---

核心职责：

```text
理解企业

预测企业

模拟企业

优化企业

规划企业
```

# 6. Causal Model

世界模型核心。

---

定义：

```text
Cause

↓

Effect
```

---

示例：

```text
库存增加

↓

库存成本增加
```

---

```text
客户满意度下降

↓

续费率下降
```

---

数据结构：

```java
public class CausalEdge {

   Node source;

   Node target;

   Double weight;

}
```
