## 五、已知 API 问题诊断结果（3 项全部定位）

| 症状 | 根因 | 前端影响 | 修复方案 |
|------|------|---------|---------|
| `/api/datalake/query` 405 | `@PostMapping` — GET 不匹配 | 前端用 POST 方式 | POST 方法即为正确 |
| `/api/twins/telemetry` 404 | 缺少路径变量 `{deviceId}` | 需 `/api/twins/{id}/telemetry` | 添加聚合端点 |
| `/api/traces` 403 | 实际路径 `/api/telemetry/traces` | 白名单无 `/api/traces/**` | 前端改路径或添加别名 |
