## Step 4: 构建 Pipeline (2 个)

### Pipeline A: 数据清洗 + 质量检查

**YAML DSL**：
```yaml
apiVersion: ecos/v2
kind: Pipeline
metadata:
  id: "pl-clean-check"
  name: "高速数据清洗与质量检查"
spec:
  execution:
    engine: memory
    timeout: 600
  nodes:
    - id: src
      type: source
      config:
        datasourceId: "${DORIS_DS_ID}"
        table: "目标表"
    - id: clean_nulls
      type: transform
      config:
        derive:
          - field: status
            expression: "coalesce(status, 'UNKNOWN')"
      dependsOn: [src]
    - id: quality_check
      type: transform
      config:
        checks:
          - rule: "not_null"
            column: "id"
            severity: "critical"
          - rule: "range"
            column: "amount"
            min: 0
            max: 99999999
            severity: "warning"
          - rule: "uniqueness"
            column: "id"
            severity: "critical"
      dependsOn: [clean_nulls]
    - id: sink
      type: sink
      config:
        datasourceId: "${DORIS_DS_ID}"
        table: "目标表_clean"
        mode: overwrite
      dependsOn: [quality_check]
```

**创建**：
```bash
curl -s -X POST http://localhost:8080/api/v1/engine/data/pipeline/tasks \
  -H "Content-Type: application/json" \
  -d "{\"name\":\"高速数据清洗与质量检查\",\"yaml_content\":\"$(cat pipeline_a.yaml | sed 's/"/\\"/g' | tr '\n' '\\n')\"}"
```

### Pipeline B: 聚合分析 + 质量评分

**YAML DSL**：
```yaml
apiVersion: ecos/v2
kind: Pipeline
metadata:
  id: "pl-agg-score"
  name: "高速数据聚合与质量评分"
spec:
  execution:
    engine: memory
    timeout: 600
  nodes:
    - id: src
      type: source
      config:
        datasourceId: "${DORIS_DS_ID}"
        table: "目标表"
    - id: filter_valid
      type: transform
      config:
        expression: "filter(notnull(amount) and amount > 0)"
      dependsOn: [src]
    - id: agg
      type: aggregate
      config:
        groupBy: ["status"]
        aggregations:
          - field: total_amount
            function: sum
            sourceField: amount
          - field: avg_amount
            function: avg
            sourceField: amount
          - field: record_count
            function: count
            sourceField: id
      dependsOn: [filter_valid]
    - id: score
      type: transform
      config:
        derive:
          - field: quality_score
            expression: "round(record_count * 100 / (record_count + nullif(0, 0)), 2)"
      dependsOn: [agg]
    - id: sink
      type: sink
      config:
        datasourceId: "${DORIS_DS_ID}"
        table: "质量评分结果"
        mode: overwrite
      dependsOn: [score]
```

## Step 6: 执行 Pipeline + 质量规则

### 6.1 执行 Pipeline A

```bash
curl -s -X POST "http://localhost:8080/api/v1/engine/data/pipeline/tasks/${PL_A_ID}/run" \
  | python3 -c "import sys,json; d=json.load(sys.stdin); print(f'runId={d[\"data\"][\"runId\"]}')"
```

### 6.2 轮询执行状态

```bash
run_id="xxx"
while true; do
  status=$(curl -s "http://localhost:8080/api/v1/engine/data/pipeline/runs/$run_id" \
    | python3 -c "import sys,json; print(json.load(sys.stdin)['data']['status'])")
  echo "Status: $status"
  [ "$status" = "SUCCEEDED" ] || [ "$status" = "FAILED" ] && break
  sleep 2
done
```

### 6.3 查看质量报告

```bash
curl -s "http://localhost:8080/api/v1/engine/data/pipeline/runs/$run_id/steps" \
  | python3 -c "
import sys,json
d=json.load(sys.stdin)
for step in d.get('data',[]):
    print(f'{step[\"node_id\"]}: {step[\"status\"]} rows_in={step[\"rows_input\"]} rows_out={step[\"rows_output\"]} elapsed={step[\"elapsed_ms\"]}ms')
"
```
