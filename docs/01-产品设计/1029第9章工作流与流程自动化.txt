# 第9章 Workflow & Process Automation（工作流与流程自动化平台）

（ECOS Detailed Functional Design Specification）

---

# 9.1 模块概述

## 模块名称

```text
Workflow & Process Automation Platform
```

---

## 模块定位

ECOS业务流程执行中心。

负责：

```text
流程编排

审批流

任务流

人机协同

Case管理

Mission执行
```

---

对应：

```text
Palantir Foundry Workflow Builder

Operational Workflow

Case Management

Human-in-the-loop
```

---

## 在ECOS中的位置

```text
Ontology

↓

Object Runtime

↓

Action & Rule

↓

Workflow

↓

Agent

↓

Mission Control
```

---

## 核心目标

实现：

```text
业务对象

↓

动作(Action)

↓

流程(Workflow)

↓

任务(Task)

↓

人员/Agent

↓

结果(Result)
```

# 9.26 Workflow API

---

创建流程：

```http
POST /api/v1/workflows
```

---

发布流程：

```http
POST /api/v1/workflows/publish
```

---

启动流程：

```http
POST /api/v1/workflows/start
```

---

查询实例：

```http
GET /api/v1/workflows/instances/{id}
```

---

终止流程：

```http
POST /api/v1/workflows/terminate
```
