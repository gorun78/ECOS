# 4.1 模块概述

## 模块名称

```text
Storage & Lakehouse Platform
```

---

## 模块定位

ECOS统一数据底座。

负责：

```text
数据存储

数据管理

数据访问

数据生命周期

统一计算访问
```

对应Foundry：

```text
Foundry Data Lake

Ontology Storage

Object Storage

Data Warehouse
```

---

## 核心目标

构建企业统一数据空间：

```text
Structured Data

+

Semi Structured Data

+

Unstructured Data

+

Graph Data

+

Vector Data
```

---

## 功能架构

```text
Storage Platform

├── Object Storage
├── Lakehouse Storage
├── Data Warehouse
├── Vector Storage
├── Graph Storage
├── Cache Layer
```

---

```text

```

---

```text

```

---

## Business Layer

```text

```

---

形成：

```text

```

---

## Ontology Layer

形成：

```text

```
